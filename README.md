# Todo Studio

간단한 정적 Todo 앱입니다.

## 기능

- 할 일 추가
- 완료 체크
- 전체 / 진행 / 완료 필터
- 완료 항목 삭제
- 브라우저 로컬 저장

## GitHub Pages 배포

이 저장소를 GitHub에 올린 뒤 `Settings > Pages`에서 다음처럼 설정하세요.

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/(root)`

저장 후 몇 분 뒤 `https://사용자명.github.io/저장소명/` 주소로 접속할 수 있습니다.
